# 星芒集盒

一款集成了多种实用工具的微信小程序，提供便捷的日常工具和娱乐功能。

## ✨ 特性

- 🔐 **微信授权登录** - 支持微信一键登录，安全便捷
- 🌓 **深色模式** - 支持跟随系统或手动切换深色/浅色主题
- 📱 **全面适配** - 完美适配各种机型和安全区域（刘海屏）
- 🔄 **自动更新** - 检测并提示应用更新
- 🎨 **精美UI** - 现代化设计，用户体验优秀

## 🚀 快速开始

### 环境要求

- 微信开发者工具
- 微信小程序基础库 2.x 或更高版本

### 安装运行

1. 克隆或下载本项目
```bash
git clone [项目地址]
cd 星芒集盒_微信小程序
```

2. 使用微信开发者工具打开项目根目录

3. 在开发者工具中点击"编译"即可运行

## 🎨 主题系统

本项目支持完整的深色模式系统：

- **跟随系统** - 自动适配系统深色/浅色模式
- **手动切换** - 用户可在设置中手动选择主题

主题变量定义在 `app.wxss` 中，所有页面使用 CSS 变量实现主题切换。
